# 🧪 Lab 4: Field Service Operations

## 🎯 Objective
Manage field service requests and scheduling.

---

## 🔧 Steps

- Create a Case (service request)
- Convert Case → Work Order

---

## 👷 Assign Resource

- Assign a Field Engineer
- Schedule service visit

---

## 📍 Execution

- Update work order status:
  - Scheduled → In Progress → Completed

---

## 🤖 Automation Task

- Send customer notification when:
  - Work order is scheduled

---

## ✅ Expected Result

- Work order completed
- Engineer assigned
- Customer notified
